# HTTP Header Scanner

## Overview
A simple tool to scan HTTP headers for missing security features.

## Features
- Fetches HTTP headers from a target URL.
- Identifies missing security headers like:
  - Strict-Transport-Security
  - Content-Security-Policy
  - X-Frame-Options
